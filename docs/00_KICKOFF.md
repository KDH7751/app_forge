# Kickoff

## Identity

This repository exists to produce a reusable Flutter app engine template, not a
single toy application.

## Success Conditions

- keep `lib/engine/` reusable across apps
- configure the app in only three files under `lib/app/`
- extend the app by adding feature slices under `lib/features/`

## Working Style

- architecture first, then feature work
- change code in small batches
- AI proposals must remain explainable and reviewable

## Hard Rules

- do not copy company code or proprietary assets
- do not let engine know app or features
- do not bypass the coding contract for convenience
